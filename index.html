📦 Projeto Principal

Invexus - Sistema de Gestão de Estoque

● Next.js 15 🌿 TypeScript 💙 Tailwind CSS 🔥 Firebase

Sistema web empresarial de alta performance para controle de ciclo completo de produtos. Focado em escalabilidade e precisão para múltiplas empresas simultâneas.

Principais Funcionalidades

  • Gestão Multi-tenant (Múltiplas empresas e planos)
  • Controle de Lotes com Sistema FIFO Automático
  • Cálculo de Custo Médio Ponderado e Rastreabilidade
  • Dashboard de Métricas com Alertas de Estoque Baixo
  • Gestão de Equipes com Permissões Granulares
  • Transações Atômicas no Firestore para integridade de dados
